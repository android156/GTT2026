\# INSTR 11: Приёмка и чек-лист



\## URL/SEO

\- /catalog/<...>/ всегда 301 на /<...>/

\- запросы без завершающего слеша 301 на вариант со слешем

\- canonical на новых URL

\- sitemap.xml содержит только новые URL

\- robots.txt запрещает /admin/



\## Функции

\- админка доступна, CRUD работает

\- каталог 4 уровня работает

\- типоразмер имеет JSON-LD Product

\- новости работают

\- формы лидов работают

\- редиректы из админки применяются с приоритетом над системными



\## Запуск

\- python app.py поднимает сайт

\- есть команды миграций и инструкция в README
